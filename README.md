# Learn-OpenGL

This repository contains the entirety of the code i have authored during my dedicated study of OpenGL Core, utilizing the educational resources available on the website https://learnopengl.com.

with this repository, you will find a compilation of tutorial codes and various self imposed exercises that i have undertaken. it is important to note that this project is a work in progress, and continuous development and refinement are still ongoing.

# How to use
1. clone or download as zip the source code of this project
2. download cmake minimum of version 3.25
3. build the project with cmake and that is all 

you can also open this project as a cmake folder in IDEs such as CLion and Visual Studio.
